# 🛒 NovaCart

> A modern full-stack e-commerce platform built to deliver a seamless online shopping experience with scalable architecture, responsive design, and AI-assisted development workflows.

![NovaCart Banner](./assets/banner.png)

## 🚀 Overview

NovaCart is a production-inspired e-commerce platform designed to simulate a real-world online marketplace. It enables users to discover products, manage shopping carts, save favorites, track orders, and enjoy a smooth shopping experience across all devices.

The project was developed using modern web technologies and AI-assisted development tools to accelerate design and implementation while following industry-standard practices.
